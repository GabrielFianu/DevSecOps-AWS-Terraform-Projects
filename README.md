# DevSecOps-AWS-Terraform-Projects

## Project Overview
This project demonstrates a CI/CD pipeline using AWS CodeCommit, CodeBuild, CodePipeline, SonarCloud, Snyk, and OWASP ZAP.

### Pipeline Flow
1. CodeCommit: Source code repository
2. CodeBuild: Automated build and testing
3. SonarCloud: Code quality analysis
4. Snyk: Vulnerability scanning
5. OWASP ZAP: Security testing
6. CodePipeline: Continuous deployment

Setup Instructions
Connect GitBash with AWS CodeCommit
Integrate with SonarCloud
Integrate with Snyk
Integrate with OWASP ZAP

### Example Use Cases
- Push code changes to CodeCommit
- Trigger CodePipeline deployment
- Monitor SonarCloud, Snyk, and OWASP ZAP reports
